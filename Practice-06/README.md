# Practice 6: Python File Handling and Built-in Functions

This practice covers file operations, directory management, and core built-in functions.

## Repository Structure

```
Practice-06/
├── file_handling/
│   ├── read_files.py          # Reading files: read(), readline(), readlines()
│   ├── write_files.py         # Writing, appending, creating files
│   └── copy_delete_files.py   # Copying with shutil, deleting with os
├── directory_management/
│   ├── create_list_dirs.py    # mkdir, makedirs, listdir, rmdir
│   └── move_files.py          # Moving and copying files between dirs
├── builtin_functions/
│   ├── map_filter_reduce.py   # map(), filter(), reduce()
│   └── enumerate_zip_examples.py  # enumerate(), zip(), sorted(), len(), sum()
└── README.md
```

## Topics Covered

- **File handling**: read, write, append, create, copy, delete
- **Directory management**: mkdir, makedirs, listdir, rmdir, move
- **Built-in functions**: map, filter, reduce, enumerate, zip, sorted, len, sum, min, max
