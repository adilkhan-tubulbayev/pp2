# Practice 3: Python Functions and OOP

This practice covers Python functions, lambda expressions, classes, and inheritance.

## Repository Structure

```
Practice-03/
├── functions/
│   ├── basic_functions.py      # Function definition and calling
│   ├── function_arguments.py   # Positional, keyword, default arguments
│   ├── return_values.py        # Return values and recursion
│   └── args_kwargs.py          # *args and **kwargs
├── lambda/
│   ├── lambda_basics.py        # Lambda syntax and usage
│   ├── lambda_with_map.py      # Lambda with map()
│   ├── lambda_with_filter.py   # Lambda with filter()
│   └── lambda_with_sorted.py   # Lambda with sorted()
├── classes/
│   ├── class_definition.py     # Class creation and objects
│   ├── init_method.py          # The __init__() method
│   ├── class_methods.py        # Instance methods and __str__()
│   └── class_variables.py      # Class vs instance variables
├── inheritance/
│   ├── inheritance_basics.py   # Parent and child classes
│   ├── super_function.py       # Using super()
│   ├── method_overriding.py    # Method overriding and polymorphism
│   └── multiple_inheritance.py # Polymorphism without inheritance
└── README.md
```

## Topics Covered

- **Functions**: Definition, arguments, return values, *args/**kwargs
- **Lambda**: Syntax, map(), filter(), sorted()
- **Classes**: Definition, __init__, methods, class/instance variables
- **Inheritance**: Parent/child classes, super(), method overriding, polymorphism
