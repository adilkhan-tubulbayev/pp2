# Practice 4: Python Advanced Topics

This practice covers iterators, generators, dates, math operations, and JSON handling.

## Repository Structure

```
Practice-04/
├── generators.py              # Iterator and generator exercises
├── dates.py                   # Date and time operations
├── math_exercises.py          # Math and area calculations
├── json_exercises.py          # JSON parsing and display
├── exercices/
│   ├── generators.md          # Generator task descriptions
│   ├── date.md                # Date task descriptions
│   ├── math.md                # Math task descriptions
│   └── json/
│       ├── README.md          # JSON task description
│       └── sample-data.json   # Sample data for JSON exercise
└── README.md
```

## Topics Covered

- **Generators**: yield, generator functions, generator expressions
- **Dates**: datetime, timedelta, date arithmetic
- **Math**: degree/radian conversion, area calculations, math module
- **JSON**: parsing JSON files, formatted output
