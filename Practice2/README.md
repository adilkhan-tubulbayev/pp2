# Practice 2: Python Control Flow Basics

This practice covers Python control flow structures including conditionals and loops.

## Repository Structure

```
Practice2/
├── boolean/
│   ├── boolean_intro.py      # Boolean values and bool() function
│   ├── boolean_comparison.py # Comparison operators
│   └── boolean_operators.py  # Logical operators and isinstance()
├── if_else/
│   ├── if_statement.py       # Basic if statements
│   ├── if_else.py            # If-else statements
│   ├── if_elif_else.py       # If-elif-else chains
│   └── short_hand_if.py      # Ternary operators
├── loops/
│   ├── while_loop.py         # While loops
│   ├── while_break.py        # Break in while loops
│   ├── while_continue.py     # Continue in while loops
│   ├── for_loop.py           # For loops and range()
│   ├── for_break.py          # Break in for loops
│   └── for_continue.py       # Continue in for loops
└── README.md
```

## Topics Covered

- **Boolean**: Values, comparisons, operators
- **Conditionals**: if, else, elif, shorthand/ternary
- **Loops**: while, for, break, continue
