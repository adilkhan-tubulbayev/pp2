# Practice 5: Python Regular Expressions (RegEx)

This practice covers regular expressions in Python using the `re` module, including receipt parsing.

## Repository Structure

```
Practice-05/
├── regex_exercises.py   # 10 regex exercises
├── receipt_parser.py    # Receipt parsing from raw.txt
├── raw.txt              # Sample receipt data
├── regex.md             # Exercise descriptions
└── README.md
```

## Topics Covered

- **RegEx basics**: search, findall, split, sub, match
- **Metacharacters**: `.`, `*`, `+`, `?`, `^`, `$`, `[]`, `|`, `()`
- **Special sequences**: `\d`, `\w`, `\s`, `\D`, `\W`, `\S`
- **Practical**: parsing receipt data, extracting prices, products, dates
